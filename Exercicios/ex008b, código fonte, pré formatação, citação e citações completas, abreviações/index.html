Código fonte/Pré formatação

Se usa a tag Code para indicar ao navegador que estou escrevendo um código.
Exemplo: O comando document.GetElementById('teste') é escrito em linguagem JavaScript

Se usa a tag Pre para formatar o código escrito da forma que está aqui onde estou escrevendo ele e passar exatamente dessa forma para o navegador

         Exemplo:
num = int(input('digite um numero'))
if num % 2 ==0:
    print(f'O {num} é par')
else
    print(f' O {num} é impar')
print('Fim do programa')
